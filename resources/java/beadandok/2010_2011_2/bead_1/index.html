A feladat leírása

Készíts egy média értékelõ programozói könyvtárat! A program 3 különbözõ médiát tudjon kezelni: könyvet, újságot, valamint DVD-t. A médiáknak általánosan van címe és példányszáma, valamint könyvnek szerzõje, újságnak szerkesztõje, DVD-nek pedig rendezõje.

Készíts egy általános értékelés osztályt! Ezt a programban médiákra specializálva lehessen használni. Egy értékeléshez tartozik egy egyedi azonosító (long), egy tetszõleges média, valamint egy 1-5 között lévõ pontszám. A program ilyen értékelések kezelésére nyújtson egy interfészt, az alábbi funkcionalitásokkal:

  1. Értékelések listázása
  2. Új értékelés felvétele
  3. Meglévõ értékelés módosítása
  4. Meglévõ értékelés törlése
  5. Legjobb értékelés megkeresése
  6. Legrosszabb értékelés megkeresése

Az értékelések tárolását tetszõleges adatszerkezettel megoldhatod.

Értékelések listázásánál vissza kell adni egy listát, amely tartalmazza az eddig rögzített, esetlegesen módosított, nem törölt elemek listáját. A sorrend tetszõleges. Kell egy külön függvény is, amely a nyilvántartott értékelések számát adja meg.

Új értékelés felvételénél be kell kérni a média típusától függõ adatokat, majd felvenni egy új értékelést.

Meglévõ értékelés módosításánál az értékelés egyedi azonosítójának függvényében meg kell tudni változtatni az értékelés pontszámát. A paraméterrõl ellenõrzendõ, hogy a megszabott intervallumban van-e.

Meglévõ értékelés törlésénél az értékelés egyedi azonosítójának függvényében törölni kell az adott értékelést a nyilvántartott listából.

A legjobb (legrosszabb) értékelés megkeresése esetén a program keresse és adja meg a legjobb (legrosszabb) értékelést, ami jelenleg az adatbázisban található.

Technikai részletek

  • Az értékelések osztályának equals() függvénye az egyedi ID alapján hasonlítsa össze az objektumokat.
  • Az alkalmazás implementálja a tesztfájlban található IMediaManager interfészt!
  • A média osztályok közös õse legyen az AMedia absztrakt osztály.
  • A használandó konstruktorok szignatúrája a paraméterek nevével:
    • Newspaper(String title, int copies, String editor)
    • Book(String title, int copies, String author)
    • DVD(String title, int copies, String director)
    • Rating(long id, AMedia media, int score)
  • A toString() függvény felüldefiniálása nem kötelezõ sehol, ám a hibaüzenetek úgy lesznek konstruktívak, ha ezek megvannak.
